# README

Ruby on Rails blog application using ActionCable for realtime chat/response.

## Installation

```bash
git clone https://github.com/aarias89/rails-app.git
```

## Run Locally

```bash
bundle install
```

```
rails db:create
```

```
rails db:migrate
```

```
rails s -p 3001
```
